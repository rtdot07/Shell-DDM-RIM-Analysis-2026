# Shell-PLC-Equity-Valuation-Analysis (Financial Modelling & Projections till 2031)
## Project Objective
The objective of this project is to perform a comprehensive equity valuation of Shell PLC using two fundamental valuation techniques: the Dividend Discount Model (DDM) and the Residual Income Model (RIM). This analysis projects financial performance through 2031 to determine if the stock is currently undervalued or overvalued in the market.
